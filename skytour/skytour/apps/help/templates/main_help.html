Skytour

Overview

This software is to help you:

  • Plan Observing Sessions
  • Record/Log Observations
  • Add Images Your Taken

It is designed for observing in several ways (modes):

  • Naked-Eye
  • Binoculars
  • Small Telescopes (≤ 6")
  • Medium Telescopes (> 6")
  • Imaging ("Smart") Telescopes

It comes with an extensive library of several thousand Deep-Sky Objects (DSOs) and Solar-System Objects (SSOs). You can add comets and asteroids.

If you travel with your scopes, you can add as many observing locations as you want.

How it basically works

Setting the Cookie

Everything operates against a "cookie" where you set:

  • Your Location
  • The UT Date/Time
  • What Mode You'll be Using (mainly)
The cookie sorts out "what's up when", including the location of the Planets, and any Asteroids/Comets you've included.

Once set, you can:

  • See what's up right now at your location
  • See what's up at the date/time of the cookie
  • Create an Observing "plan" (for printing) - this is helpful if you're not going have laptop access while observing
    • .

Observing Sessions

When you start a new Observing Session you can log every object you observe, as well as observing conditions (weather, sky brightness - if you have a light meter). Skytour keeps track of all that logging for you. It you're using an optical telescope, you can also record the eyepieces and filters used.

If you're using an imaging "smart" scope you can upload images of the objects you've observed.
