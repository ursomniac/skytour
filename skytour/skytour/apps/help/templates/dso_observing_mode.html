Observing Modes

  • N = Naked Eye
  • B = Binoculars
  • S = Small Telescope (≤ 6")
  • M = Medium Telescope (> 6")
  • I = Imaging Telescope

Each observing mode has a priority and a viability.

Priority

This is a value from 0 to 4 and roughly suggests which object are "must see"/

  • 0 = Extremely Low
  • 1 = Low
  • 2 = Medium
  • 3 = High
  • 4 = Highest

Viability

This value is from 0 to 10, and indicates the difficulty to observe the object under normal conditions using each mode of observing.

  • 0 = Not Viable
  • 1 = Unlikely Viable
  • 2 = Extreme Difficulty
  • 3 = Difficult
  • 4 = Challenging
  • 5 = Requires Patience
  • 6 = Generally Visible
  • 7 = Usually Viable
  • 8 = Easy
  • 9 = Very Easy
  • 10 = Extremely Easy
