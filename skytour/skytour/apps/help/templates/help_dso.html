Deep Sky Objects - DSOs

The DSO Catalog

Skytour comes seeded with a table of over 3,000 deep-sky objects.

This list is somewhat complete, in that it tries to incorporate objects in both with hemispheres down to approximately 13th magnitude, which should suffice for most small/medium telescopes as well as the imaging/"smart" telescopes currently on the market.

DSO Fields

To avoid crowding and taking into consideration that in many cases more than one DSO can appear in a telescope's (wide) field of view, there's a related "DSOInField" table of - i.e., over 2,000 additional DSOs that are adjacent to (usually brighter) DSOs. (So the total is just over 6,000.)

Thus going to a "DSO page", for example, M 106 the record contains not only the bright galaxy, but also 3 other nearby galaxies. In general, all the galaxies brighter than magnitude 13.0 within ~45 arcminutes of the "primary" DSO will be included.

Occasionally two adjacent objects might be held as separate records: for example, Caldwell 60 and 61 (the "Antenna Galaxies") have separate entries, even though their fields overlap.

Using the Search box in the top navigation WILL scan both lists: in other words, searching for "NGC 4232" will return the page for "M 106" because the former is in the field of the latter.

DSO Metadata

Where possible each DSO has the following metadata:
  • Coordinates (J2000)
  • Magnitude (typically V band)
  • Angular Size
  • Surface Brightness (in mag/arcmin2)
  • Distance (units vary based on scale)
  • Object Type (galaxy, nebula, cluster, etc.)
  • Morphological Type - varies by object type:
    • Galaxies - Shapley classfication, e.g., SAc
    • Globular Clusters - Harlow/Hogg (from I to XII)
    • Open Clusteres - Trumpler, e.g., II 2 r
    • Dark Nebulae - Barnard opacity
    • Bright Nebulae - Sharpless, e.g, 2 1 2
    • Planetary Nebulae - Vorontsov-Velyaminov, e.g., 4+3a
    • Orientation Angle - degrees E from N
    • Contrast Index - obtained through Stellarium - not sure how to use it

Metadata Sources

While the DSO and DSOInField models have fields for all of the above, seeding of the database tables pulls from SIMBAD and HyperLeda (mostly for galaxies). On display, the code tries them in the order HyperLeda, SIMBAD, and what in the Admin; you can override this to use just the Admin in the DSO records (on a case-by-case basis).

Object Types

DSO Observing Modes

Each DSO (and it's associated "field") is assign one or more observing "modes" related to how appropriate it might be to different observing situations:

  • N: Naked-Eye - objects that don't require optical aid to detect, presuming mostly dark (no Moon) conditions at a relatively dark location. Typically these objects will be 6th magnitude or brighter.
  • B: Binoculars - objects that are somewhat large (several arcminutes) and bright (< magnitude ~7.5)
  • S: Small Telescope - with aperture ≤ 6" - objects down to about magnitude 11, or - in a few cases - fainter but with high surface brightness.
  • M: Medium Telescope - with aperture > 6" - objects down to magnitude 12 (sometimes fainter)
  • I: Imaging Telescope - mostly objects with magnitude < 15, but also excluding those with very small angular size - usually 2' in diameter, however exceptions have been made for some even smaller objects - down to ~30' because post-processing of image stacks have been somewhat successful in showing structure.

DSO Observing Priorities

For each mode assigned a "priority" has been set:

  • 0 - Lowest - almost not worth attempting
  • 1 - Low - doable, but not likely to be especially interesting
  • 2 - Medium - less well-known, generally smaller or fainter or unlikely to show much detail
  • 3 - High - definitely worth attempting
  • 4 - Highest - the "must see" objects on anyone's list

These are all arbitrary.

On many pages, the Imaging Mode Priority is shown (encircled number with color) to make it easy to scan lists if you're using an imaging/"smart" telescope. For the other modes, the format is the priority value and label, e.g., "3 - High".

DSO Observing Viability

In addition, there is also a "viability" assigned to each mode for a DSO. These range from 0 to 10, and crudely (and arbitrarily) say how difficult it would be to observe (or even detect) an object:

  • 0 - Not Viable
  • 1 - Unlikely Viable
  • 2 - Extreme Difficulty
  • 3 - (Very)) Difficult
  • 4 - Challenging
  • 5 - Requires Patience
  • 6 - Generally Visible
  • 7 - Usually Easy
  • 8 - Easy
  • 9 - Very Easy
  • 10 - Extremely Easy

Of course, what's Easy for a larger telescope might be impossible in Binoculars, so it's important to keep the observing mode in consideration.
